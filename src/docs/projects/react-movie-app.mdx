## Description

Movie app built by using The Movie Database API. I used login boilerplate from last project for login, logout, signin functions.

Logged-in users can use the favorite page, they can add their favorite movie from the movie detail pages by clicking favorite button.

### Stacks

- HTML
- CSS
- JS
- React
- Redux
- Express

##### Libraries

- Mongoose
- Axios
- Ant Design

##### Database
