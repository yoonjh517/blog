## Delpoy Gatsby website to Github pages

#### Connect gatsby blog with github pages

```
git remote add  origin https://github.com/<your-github-username>/<repository-name>.git
```

#### Make a branch for developing

```
git checkout -b dev
```

#### Push project on gitgub

```
git add .
git commit -m 'commit message'
git push -u origin dev
```

#### Install gh-pages

```
npm install gh-pages --save-dev
```

#### Set up package.json

```json title="package.json"
"scripts": {
  ...
  "deploy": "gatsby build --prefix-paths && gh-pages -d public -b main"
}
```

#### Set up gatsby-config.js

```js title="gatsby-config.js"
module.exports = {
  pathPrefix: "/blog",
  plugins: [`gatsby-plugin-offline`],
};
```

#### Deploy

```
npm run deploy
```

#### Github setting

1. Go to gitgub
1. Select repository
1. Go to settings
1. Go to pages
1. Change source to main
1. Save
